# 📘 자바의 신 VOL.1 – 5장:  계산을 하고 싶어요

## 1. 연산자 종류와 의미

- **산술 연산자**  
  `+`, `-`, `*`, `/`, `%`  
  수학 연산 처리

- **대입 연산자**  
  `=`, `+=`, `-=`, `*=`, `/=`, `%=`  
  계산 결과를 변수에 저장

- **증감 연산자**  
  `++`, `--`  
  전위/후위 연산 구분 필요

- **비교 연산자**  
  `==`, `!=`, `<`, `>`, `<=`, `>=`  
  비교 결과는 boolean

- **논리 연산자**  
  `&&`, `||`, `!`  
  조건 분기용 연산

- **삼항 연산자**  
  `조건 ? 값1 : 값2`  
  조건식 결과에 따라 값 선택

## 2. 형 변환 (Casting)

- **자동 형변환 (Widening)**  
  작은 타입 → 큰 타입으로 자동 변환

- **명시적 형변환 (Narrowing)**  
  큰 타입 → 작은 타입으로 명시적 캐스팅 필요  
  데이터 손실 가능성 있음
